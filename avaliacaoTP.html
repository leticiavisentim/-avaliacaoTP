O que é Git e github

Github é um site para guardar códigos, também serve como portfólio para desenvolvedores. Git é a versão desktop, que facilita na hora de salvar em núvem, os códigos feitos.

Versão do sistema é o mesmo que versionar código?

Não. Um código pode ter várias versões para ser ajustado. Mas a versão do sistema pode ser apresentado depois de vários códigos versionados.
